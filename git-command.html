GIT COMMANDS ------------ Configuration Configure your name $ git config --global user.name "Your Name" Configure your username $ git config --global user.username githubusername Configure your email $ git config --global user.email youremail@example.com Check in Check status of changes to a repository $ git status View changes between commits $ git diff Add a file's changes to be commited $ git add To add all files' changes $ git add . To commit (aka save) the changes you've added with a short message describing the changes $ git commit -m "your commit message" Remotes To push your changes to a remote $ git push To pull in changes from remote $ git pull To set a URL for a remote $ git remote set-url Add a remote $ git remote add Branches Merge a branch into current branch $ git merge Change the branch you're working on $ git checkout Delete a local branch $ git branch -d Delete a remote branch $ git push --delete BASH COMMANDS ------------- Make a new folder (aka directory) $ mkdir Navigate into an existing folder (aka change directory) $ cd Back out of a folder/directory $ cd .. Back out all the way to the root directory $ cd List the items in a folder $ ls Remove (delete) a file $ rm Remove (delete) a folder $ rm -rf
